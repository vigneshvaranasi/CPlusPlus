Files

File handling in C++ allows you to work with files on the system, enabling reading from and writing to files. The <fstream> header in C++ provides classes like ifstream, ofstream, and fstream to perform file operations.

Steps of File Operations

  1. Include Necessary Headers:
    • #include <iostream>
    • #include <fstream>
    • #include <string>
  1. Opening a File:
    • For Writing (Output): ofstream.objectName("example.txt");
    • For Reading (Input): ifstream.objectName("example.txt");
    • For Both Reading and Writing: fstream.objectName("example.txt",ios::in|ios::out);
  1. Check if the File is Open:
    • if (objectName.is_open()) { /* File is open for writing */ } else { /* Handle error */ }
  1. Writing to a File:
    • objectName << "Hello, File Handling in C++!";
  1. Reading from a File:
    • string content;
    • while (getline(objectName,content)) { /* Process each line */ }
  1. Appending to a File:
    • objectName.open("example.txt",ios::app);
    • objectName << "Appending content to the file\n";
  1. Closing the File:
    • objectName.close();

File Opening Modes

Mode NameDescriptionSyntax
ios::inOpens the file for reading.ifstream objectName("example.txt",ios::in);
ios::outOpens the file for writing. Truncates existing content.ofstream objectName("example.txt",ios::out);
ios::appOpens the file for writing, appending to existing content.ofstream objectName("example.txt",ios::app);
ios::binaryOpens the file in binary mode (for binary data).ifstream binaryInput("binary.dat",ios::binary);
ios::atePositions the file pointer at the end upon opening.fstream objectName("example.txt", ios::in);
ios::truncTruncates existing content if the file is opened for writing.ofstream objectName("example.txt",ios::trunc);

These modes can be combined using the bitwise OR (|) operator when opening a file with both input and output modes

End of a File

The eof() function returns the non-zero value, when the end of the file is detected, otherwise, it is zero.

    if (objectName.eof()) {
        cout << "Reached the end of the file." << endl;
    }

Checking for Errors during File Opening

Open File:

  • Use the is_open() method of the file stream to check if the file has been successfully opened.

    if (!objectName.is_open()) {
        cerr << "Unable to open the file." << endl;
		}
  • Always close the file stream when you're done with file operations using the close() method.
  • Utilize the cerr stream to output error messages to the standard error stream.

Pointer NameDescriptionSyntax for Input File (ifstream)Syntax for Output File (ofstream)

putPut pointer for writing charactersN/A (Not applicable for input streams)objectName.put()
tellgGet current get pointer position (input stream)objectName.tellg()N/A (Not applicable for output streams)
tellpGet current put pointer position (output stream)N/A (Not applicable for input streams)objectName.tellp()
seekgSet the get pointer position (input stream)objectName.seekg(offset, mode)N/A (Not applicable for output streams)

  • offset: Number of characters to move.
